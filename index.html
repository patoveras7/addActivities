Camino a ser Henry!

Por Veras Carrara Patricio

Mi nombre es Patricio Raul Veras Carrara (pato), tengo 30 años de edad y acabo de unirme a este desafio de la mano de Henry. Por qué decidí estudiar programación?? Actualmente me enuentro graduado de la Universidad Nacional de Córdoba y de la Universidad Siglo XXI habiendo obtenido los titulos de Abogado y Escribano, los cuales lamentablemente en Argentina no son bien pagados en la mayoria de los casos y pese a haber aprobado un concurso y haber ingresado a trabajar al Poder Judicial de la Provincia de Córdoba en dicha estructura laboral he encontrado un techo que no desaparece aunque uno se esfuerce al maximo y aunque uno sea el primero en llegar y el ultimo en irse. Por ello es que tome la decisión de ampliar mis conocimientos en otro rubo tomando contacto con varios lenguajes de programación hace algunos meses ya que siempre se me dió bien el Inglés y jamas he tenido (ni voy a tener) problemas para sentarme a estudiar, para asi quizas en algun futuro poder trabajar para una empresa que realmente valore el trabajo duro y por qué no lograr conectar el codigo con el derecho.

Cuáles son las tecnologías que mas llaman mi atención??

Javascript

Lenguaje de programación FullStack.

Se puede combinar con las siguientes tecnologias:
CSS REACT Node Js.

JavaScript es tan utilizado porque es un lenguaje versátil, potente y ubicuo que se adapta bien a una amplia gama de aplicaciones web y permite a los desarrolladores crear experiencias interactivas y dinámicas en la web. Principalmente destinado al Frntend pero pudiendo ser conectado al BackEnd a traves de la platarfoma Node Js.

REACT

Biblioteca de JavaScript destinada a estlización Frontend

Se puede combinar con las siguientes tecnologias:
CSS HTML Javascript

React es una biblioteca de JavaScript de código abierto desarrollada por Facebook que se utiliza para construir interfaces de usuario (UI) interactivas y dinámicas para aplicaciones web. Se enfoca en la creación de componentes reutilizables que representan partes específicas de la interfaz de usuario de una aplicación.

Java

Lenguaje de programación BackEnd.

Se puede combinar con las siguientes tecnologias:
Spring Framework Apache Struts Play Framework

Java se utiliza en una variedad de aplicaciones, incluyendo aplicaciones de escritorio, aplicaciones web, aplicaciones móviles (Android), sistemas embebidos, servidores de aplicaciones, sistemas de información empresarial, entre otros. Es un lenguaje muy popular tanto en la industria del desarrollo de software como en la academia, y su demanda sigue siendo alta en el mercado laboral.

Node Js.

Plataforma de trabajo para Javascript

Se puede combinar con las siguientes tecnologias:
CSS HTML Javascript.

Node.js es una plataforma de tiempo de ejecución de JavaScript de código abierto y basada en el motor V8 de Google Chrome. Permite ejecutar código JavaScript del lado del servidor, lo que significa que puedes utilizar JavaScript para construir aplicaciones de servidor y scripts de línea de comandos.
